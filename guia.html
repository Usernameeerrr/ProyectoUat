Implementación en Diferentes Lenguajes

Aprende a implementar algoritmos en lenguajes comunes. Cada ejemplo usa el algoritmo de Suma como base, pero aplica a otros.

Python (Fácil y legible)

Ventajas: Sintaxis simple, ideal para prototipos. Desventajas: Más lento para apps grandes.

# Suma de dos números
a, b = map(int, input().split())
print(a + b)

JavaScript (Para web interactiva)

Ventajas: Corre en browsers, asíncrono. Desventajas: Tipado débil puede causar errores.

// Suma de dos números
let input = prompt("Ingresa dos números:");
let nums = input.split(" ").map(Number);
console.log(nums[0] + nums[1]);

Java (Orientado a Objetos)

Ventajas: Portátil, fuerte en enterprise. Desventajas: Verboso para scripts simples.

import java.util.Scanner;
public class Suma {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int a = sc.nextInt(), b = sc.nextInt();
        System.out.println(a + b);
    }
}

C++ (Alto rendimiento)

Ventajas: Rápido, control de memoria. Desventajas: Complejo para principiantes.

#include <iostream>
using namespace std;
int main() {
    int a, b;
    cin >> a >> b;
    cout << a + b;
    return 0;
}

Algoritmos Básicos

Conceptos clave: Un algoritmo es un conjunto de pasos finitos para resolver un problema. Complejidad: O(n) para suma lineal. Tip: Siempre considera casos edge (e.g., números negativos).

Ejemplo: Algoritmo de Suma de Dos Números (Pseudocódigo en PSeInt).

Algoritmo Suma
    Escribir "Ingrese a: "; Leer a;
    Escribir "Ingrese b: "; Leer b;
    suma <- a + b;
    Escribir "Suma: ", suma;
FinAlgoritmo

Algoritmos Intermedios

Avanza con complejidad O(log n) o O(n log n). Tip: Usa divide y vencerás para optimización.

Ejemplo: Algoritmo de Búsqueda Binaria (Pseudocódigo).

Algoritmo BusquedaBinaria
Dimension arreglo[10]; // Array ordenado
Escribir "Ingrese target: "; Leer target;
inicio <- 1; fin <- 10;
Mientras inicio <= fin Hacer
    medio <- (inicio + fin) / 2;
    Si arreglo[medio] = target Entonces
        Escribir "Encontrado en ", medio;
        Salir;
    Sino Si arreglo[medio] < target Entonces
        inicio <- medio + 1;
    Sino
        fin <- medio - 1;
    FinSi
FinMientras
Escribir "No encontrado";

Nuevo: Algoritmo de Búsqueda Lineal (explicación: Recorre array hasta encontrar elemento).

Algoritmo BusquedaLineal
Dimension arreglo[5]; // Ejemplo array
Escribir "Ingrese elemento a buscar: "; Leer x;
Encontrado <- Falso;
Para i <- 1 Hasta 5 Hacer
    Si arreglo[i] = x Entonces
        Encontrado <- Verdadero;
        Escribir "Encontrado en posición ", i;
    FinSi
FinPara
Si No Encontrado Entonces
    Escribir "No encontrado";
FinSi
FinAlgoritmo

Estructuras de Datos Básicas

Fundamentales para algoritmos eficientes. Complejidad: Arrays O(1) acceso, Linked Lists O(n) búsqueda.

  • Arrays: Almacenan elementos contiguos. Ej: int arr[5] = {1,2,3,4,5}; (C++). Tip: Usa índices para acceso rápido.
  • Listas Enlazadas: Nodos conectados. Ej: class Node { int data; Node* next; }; (C++). Tip: Ideal para inserciones frecuentes.
  • Pilas (Stacks): LIFO. Ej: push/pop. Tip: Para expresiones matemáticas.
  • Colas (Queues): FIFO. Ej: enqueue/dequeue. Tip: Para BFS en grafos.

Ejemplo en Python: Implementa una pila.

class Stack:
    def __init__(self):
        self.items = []

    def push(self, item):
        self.items.append(item)

    def pop(self):
        return self.items.pop() if not self.is_empty() else None

    def is_empty(self):
        return len(self.items) == 0

Debugging y Mejores Prácticas

Evita errores comunes en ISC. Tip: Usa print statements para rastrear variables.

  • Errores Comunes: Off-by-one en bucles, null pointers en C++, undefined en JS.
  • Herramientas: Debugger en VS Code, console.log en JS, gdb en C++.
  • Consejos: Escribe código modular, comenta, y prueba con casos edge (e.g., array vacío).
